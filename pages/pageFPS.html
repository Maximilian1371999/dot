Простейший FPS шутер в Godot: создаем за 30 минут

25 марта 2026
15 мин чтения
Иван Сидоров
Для начинающих
3D
FPS

Привет! Сегодня мы создадим самый простой шутер от первого лица в Godot. Это идеальный проект для новичков, которые только начинают знакомиться с 3D разработкой. Мы разберем всё по шагам: от создания персонажа до стрельбы по мишеням.

Не бойтесь 3D! Мы будем использовать примитивные фигуры, чтобы сосредоточиться на логике, а не на графике.

Шаг 1: Создаем основу персонажа

Первым делом создадим игрока, который будет видеть мир от первого лица. Вот пошаговая инструкция:

Создание структуры персонажа

1. Создайте новую сцену (Scene → New Scene)
2. Добавьте узел CharacterBody3D и назовите его "Player"
3. Добавьте к нему дочерний узел MeshInstance3D с кубом (это будет тело)
4. Добавьте к Player узел CollisionShape3D с формой CapsuleShape3D
5. Добавьте к Player узел Camera3D и установите его текущей камерой

Иерархия сцены Player 
Player (CharacterBody3D)
├── MeshInstance3D  # Тело игрока (куб)
├── CollisionShape3D  # Коллизия (капсула)
└── Camera3D  # Камера от первого лица

Подсказка

Чтобы сделать камеру текущей, выберите узел Camera3D и в инспекторе нажмите кнопку "Make Current". Или установите это в коде: camera.current = true

Шаг 2: Скрипт для движения игрока

Теперь добавим возможность ходить с помощью клавиш WASD. Прикрепим скрипт к узлу Player.

Скрипт движения игрока 
extends CharacterBody3D

# Скорость движения (метров в секунду)
@export var speed = 5.0
# Сила прыжка
@export var jump_velocity = 4.5

# Сила гравитации (берем из настроек проекта)
var gravity = ProjectSettings.get_setting("physics/3d/default_gravity")

func _physics_process(delta):
    # Применяем гравитацию, если персонаж не на земле
    if not is_on_floor():
        velocity.y -= gravity * delta

    # Обработка прыжка
    if Input.is_action_just_pressed("jump") and is_on_floor():
        velocity.y = jump_velocity

    # Получаем ввод с клавиатуры (WASD)
    var input_dir = Input.get_vector("move_left", "move_right", "move_forward", "move_backward")

    # Преобразуем ввод в направление движения
    # transform.basis учитывает поворот персонажа
    var direction = (transform.basis * Vector3(input_dir.x, 0, input_dir.y)).normalized()

    # Применяем движение в горизонтальной плоскости
    if direction:
        velocity.x = direction.x * speed
        velocity.z = direction.z * speed
    else:
        # Плавная остановка
        velocity.x = move_toward(velocity.x, 0, speed)
        velocity.z = move_toward(velocity.z, 0, speed)

    # Применяем движение
    move_and_slide()

Важно!

Не забудьте настроить действия ввода в Project Settings!
• move_left → клавиша A
• move_right → клавиша D
• move_forward → клавиша W
• move_backward → клавиша S
• jump → Space (пробел)

Шаг 3: Вращение камеры мышкой

Теперь добавим возможность смотреть по сторонам с помощью мыши. Это важнейшая часть FPS!

Структура для плавного вращения

Чтобы камера вращалась независимо от тела, изменим структуру:
1. Добавьте к Player узел Node3D и назовите его "Head"
2. Перетащите Camera3D в узел Head
3. Теперь камера будет вращаться вместе с Head, а тело может двигаться отдельно

Скрипт вращения мышкой 
extends CharacterBody3D

# Чувствительность мыши (можно настроить в инспекторе)
@export var mouse_sensitivity = 0.002

# Ссылка на узел Head (с камерой)
@onready var head = $Head
# Ссылка на камеру
@onready var camera = $Head/Camera3D

func _ready():
    # Захватываем мышь в окно игры
    Input.set_mouse_mode(Input.MOUSE_MODE_CAPTURED)

func _input(event):
    # Если двигаем мышь - вращаем камеру
    if event is InputEventMouseMotion:
        # Вращение по горизонтали (тело поворачивается)
        rotate_y(-event.relative.x * mouse_sensitivity)

        # Вращение по вертикали (только голова с камерой)
        head.rotate_x(-event.relative.y * mouse_sensitivity)

        # Ограничиваем угол наклона головы, чтобы не крутилась на 360
        head.rotation.x = clamp(head.rotation.x, deg_to_rad(-90), deg_to_rad(90))

func _unhandled_input(event):
    # Если нажали Escape - выходим из режима захвата мыши
    if event.is_action_pressed("ui_cancel"):
        if Input.get_mouse_mode() == Input.MOUSE_MODE_CAPTURED:
            Input.set_mouse_mode(Input.MOUSE_MODE_VISIBLE)
        else:
            Input.set_mouse_mode(Input.MOUSE_MODE_CAPTURED)

Объяснение для новичков

rotate_y() вращает весь объект (тело) по горизонтали.
head.rotate_x() вращает только голову с камерой по вертикали.
clamp() ограничивает угол, чтобы нельзя было запрокинуть голову назад на 360°.

Шаг 4: Создаем простую пулю

Теперь создадим сцену пули, которая будет лететь прямо. Это отдельная сцена, которую мы будем создавать при стрельбе.

Создание сцены пули

1. Создайте новую сцену (Scene → New Scene)
2. Добавьте узел RigidBody3D и назовите "Bullet"
3. Добавьте дочерний MeshInstance3D со сферой (это пуля)
4. Добавьте CollisionShape3D с формой SphereShape3D
5. Сохраните сцену как bullet.tscn

Скрипт пули 
extends RigidBody3D

# Скорость пули
@export var bullet_speed = 30.0
# Время жизни пули (автоматическое удаление)
@export var lifetime = 3.0

func _ready():
    # Пуля летит вперед (относительно своего поворота)
    linear_velocity = -global_transform.basis.z * bullet_speed

    # Автоматически удаляем через время
    await get_tree().create_timer(lifetime).timeout
    queue_free()

# Если пуля во что-то врезалась - удаляем
func _on_body_entered(body):
    # Не удаляем сразу при столкновении с игроком
    if body.name != "Player":
        queue_free()

Шаг 5: Добавляем оружие и стрельбу

Теперь добавим к игроку оружие, которое будет стрелять пулями при нажатии ЛКМ.

Скрипт стрельбы (добавить к Player) 
extends CharacterBody3D

# Загружаем сцену пули (установите в инспекторе!)
@export var bullet_scene: PackedScene
# Точка появления пуль (установите в инспекторе!)
@onready var gun_point = $Head/Camera3D/GunPoint

func _ready():
    # Создаем точку выстрела, если её нет
    if not gun_point:
        var new_gun_point = Marker3D.new()
        new_gun_point.name = "GunPoint"
        $Head/Camera3D.add_child(new_gun_point)
        new_gun_point.position = Vector3(0, 0, -1)  # Немного вперед от камеры
        gun_point = new_gun_point

func _input(event):
    # Если нажата левая кнопка мыши - стреляем
    if event.is_action_pressed("fire"):
        shoot()

func shoot():
    # Проверяем, загружена ли сцена пули
    if bullet_scene == null:
        print("Ошибка: сцена пули не загружена!")
        return

    # Создаем экземпляр пули
    var bullet = bullet_scene.instantiate()

    # Добавляем пулю на сцену (в тот же родитель, что и игрок)
    get_parent().add_child(bullet)

    # Устанавливаем позицию и поворот пули
    bullet.global_transform = gun_point.global_transform

    # Можно добавить небольшую случайность для разброса
    # bullet.rotate_x(randf_range(-0.02, 0.02))
    # bullet.rotate_y(randf_range(-0.02, 0.02))

    print("Выстрел!")

Настройка оружия

1. Добавьте к Camera3D узел Marker3D и назовите "GunPoint"
2. Перетащите его немного вперед (Z = -1)
3. В инспекторе у Player установите Bullet Scene = bullet.tscn

Шаг 6: Добавляем звуки (опционально)

Звуки делают игру живой. Добавим простые звуки для ходьбы и стрельбы.

Добавление звуков 
# Дополнение к скрипту Player

# Звуки (установите в инспекторе!)
@onready var shoot_sound = $ShootSound
@onready var walk_sound = $WalkSound

# Для звуков ходьбы
var is_walking = false
var walk_timer = 0.0
var walk_step_interval = 0.5  # Интервал между шагами

func _physics_process(delta):
    # ... существующий код движения ...

    # Звуки ходьбы
    if is_on_floor() and velocity.length() > 0.5:
        if not is_walking:
            is_walking = true
            play_walk_sound()

        walk_timer += delta
        if walk_timer >= walk_step_interval:
            walk_timer = 0
            play_walk_sound()
    else:
        is_walking = false

func play_walk_sound():
    if walk_sound:
        # Немного меняем высоту звука для разнообразия
        walk_sound.pitch_scale = randf_range(0.9, 1.1)
        walk_sound.play()

func shoot():
    # ... существующий код стрельбы ...

    # Проигрываем звук выстрела
    if shoot_sound:
        shoot_sound.play()
Как добавить звуковые узлы

1. Добавьте к Player узел AudioStreamPlayer3D и назовите "ShootSound"
2. В инспекторе установите звук выстрела (WAV или OGG файл)
3. Добавьте еще один AudioStreamPlayer3D и назовите "WalkSound"
4. Установите звук шагов
5. Для WalkSound установите Max Distance = 10 и Unit Size = 5

Шаг 7: Создаем мишень

Наконец, создадим мишень, которая будет исчезать при попадании пули.

Создание сцены мишени

1. Создайте новую сцену
2. Добавьте узел StaticBody3D и назовите "Target"
3. Добавьте MeshInstance3D с цилиндром (это мишень)
4. Добавьте CollisionShape3D с формой CylinderShape3D
5. Сохраните как target.tscn

Скрипт мишени 
extends StaticBody3D

# Здоровье мишени (сколько выстрелов выдержит)
@export var health = 3
# Материал при попадании
@onready var material = $MeshInstance3D.get_surface_override_material(0)

func _ready():
    # Если нет материала - создаем новый
    if material == null:
        material = StandardMaterial3D.new()
        material.albedo_color = Color.RED  # Красная мишень
        $MeshInstance3D.set_surface_override_material(0, material)

# Вызывается при попадании пули
func take_damage(damage_amount = 1):
    health -= damage_amount

    # Мигаем белым при попадании
    material.emission = Color.WHITE
    await get_tree().create_timer(0.1).timeout
    material.emission = Color.BLACK

    # Если здоровье закончилось - уничтожаем мишень
    if health <= 0:
        destroy()

func destroy():
    # Можно добавить эффект разрушения
    print("Мишень уничтожена!")

    # Исчезаем
    queue_free()

# Автоматически определяем попадание
func _on_body_entered(body):
    if "Bullet" in body.name:
        take_damage()
        # Удаляем пулю
        body.queue_free()

Как это работает

1. Пуля сталкивается с мишенью → вызывается _on_body_entered()
2. Мишень получает урон → уменьшается здоровье
3. Мишень мигает белым → визуальная обратная связь
4. Если здоровье ≤ 0 → мишень уничтожается

Что у нас получилось?

Поздравляю! Вы только что создали простой, но полностью функциональный FPS шутер в Godot. Давайте подведем итоги:

  • ✅ Игрок ходит с помощью WASD и прыгает
  • ✅ Камера вращается мышкой как в настоящем FPS
  • ✅ При нажатии ЛКМ выстреливается пуля
  • ✅ Пуля летит и сталкивается с объектами
  • ✅ Мишень получает урон и исчезает при попадании
  • ✅ Есть звуки ходьбы и стрельбы

Куда развивать проект дальше?

1. Добавьте несколько типов оружия с разной скорострельностью
2. Создайте систему подсчета очков за попадания
3. Добавьте врагов, которые стреляют в ответ
4. Создайте несколько уровней с разным расположением мишеней
5. Добавьте меню с настройками и кнопкой "Начать игру"
