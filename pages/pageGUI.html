Как сделать простой и функциональный GUI в Godot

22 января 2026
12 мин чтения

Для начинающих

Интерфейс пользователя (GUI) — это то, с чем взаимодействует игрок в первую очередь. Хороший GUI не только информирует о состоянии игры, но и делает её удобной и приятной. В этой статье я покажу, как создать основные элементы интерфейса в Godot с нуля.

GUI — это лицо вашей игры. Простой и понятный интерфейс может сделать хорошую игру отличной!

1. Шкала здоровья через ProgressBar

Создадим простую, но красивую шкалу здоровья, которая будет уменьшаться при получении урона.

Скрипт для шкалы здоровья 
extends Control

# Ссылки на элементы GUI
onready var health_bar = $HealthBar
onready var health_label = $HealthLabel

# Параметры здоровья
var max_health = 100
var current_health = 100

func _ready():
    update_health_display()

# Функция получения урона
func take_damage(damage):
    current_health = max(0, current_health - damage)
    update_health_display()

    if current_health <= 0:
        game_over()

# Функция лечения
func heal(amount):
    current_health = min(max_health, current_health + amount)
    update_health_display()

# Обновление отображения
func update_health_display():
    # Обновляем прогресс-бар
    health_bar.value = float(current_health) / max_health

    # Обновляем текст
    health_label.text = "HP: %d/%d" % [current_health, max_health]

    # Меняем цвет
    if current_health > 50:
        health_bar.set_tint_progress(Color("2ecc71"))
    elif current_health > 25:
        health_bar.set_tint_progress(Color("f39c12"))
    else:
        health_bar.set_tint_progress(Color("e74c3c"))

func game_over():
    print("Игра окончена!")

Как это работает:

ProgressBar отображает текущее здоровье игрока. При изменении здоровья полоска плавно меняет размер и цвет (зелёный → жёлтый → красный). Метод update_health_display() обновляет визуальное отображение и текст.

Ключевые моменты:

  • Значение ProgressBar от 0.0 до 1.0
  • Динамическое изменение цвета через set_tint_progress()
  • Плавные анимации через Tween (см. совет)

Совет по ProgressBar

Используйте Tween для плавного изменения значения шкалы здоровья:

var tween = Tween.new()
add_child(tween)
tween.interpolate_property(health_bar, "value",
    health_bar.value, float(current_health)/max_health, 0.3)
tween.start()

2. Счётчик монет через Label

Создадим динамический счётчик монет с анимацией при изменении значения.

Скрипт счётчика монет 
extends Control

# Ссылки на элементы
onready var coin_label = $CoinLabel
onready var coin_icon = $CoinIcon

# Количество монет
var coins = 0

func _ready():
    update_coin_display()

# Добавление монет
func add_coins(amount):
    var old_coins = coins
    coins += amount

    # Анимация изменения числа
    animate_coin_change(old_coins, coins)

    # Сохраняем в глобальную переменную
    Global.coins = coins

# Трата монет
func spend_coins(amount):
    if coins >= amount:
        var old_coins = coins
        coins -= amount
        animate_coin_change(old_coins, coins)
        Global.coins = coins
        return true
    else:
        # Недостаточно монет
        $InsufficientCoinsAnimation.play()
        return false

# Обновление отображения
func update_coin_display():
    coin_label.text = str(coins)

    # Добавляем разделители тысяч
    if coins >= 1000:
        coin_label.text = "%d" % [coins]

# Анимация изменения счёта
func animate_coin_change(from, to):
    var tween = Tween.new()
    add_child(tween)

    # Анимация числа
    tween.interpolate_method(self, "update_coin_animation",
        from, to, 0.5, Tween.TRANS_QUAD, Tween.EASE_OUT)
    tween.start()

    # Анимация иконки
    $CoinIcon/AnimationPlayer.play("coin_bounce")

    yield(tween, "tween_completed")
    tween.queue_free()

func update_coin_animation(value):
    # Промежуточное значение для анимации
    coin_label.text = str(int(value))

Как это работает:

Label отображает текущее количество монет. При изменении значения запускается анимация через Tween, которая плавно меняет число от старого к новому. Иконка монеты также анимируется при клике.

Особенности реализации:

  • Анимация изменения чисел через interpolate_method()
  • Звуковые и визуальные эффекты при сборе
  • Проверка достаточности средств перед тратой

3. Слот для предметов (Panel + TextureButton)

Создадим интерактивный слот инвентаря, в который можно помещать предметы.

Скрипт слота инвентаря 
extends Panel

# Ссылки на элементы слота
onready var item_texture = $ItemTexture
onready var amount_label = $AmountLabel
onready var highlight = $Highlight

# Данные предмета в слоте
var item_data = null
var item_amount = 0
var slot_index = 0

# Настройки слота
export var slot_size = Vector2(64, 64)
export var can_drop_items = true

func _ready():
    # Устанавливаем размер
    rect_min_size = slot_size

    # Изначально скрываем предмет
    item_texture.visible = false
    amount_label.visible = false
    highlight.visible = false

# Установка предмета в слот
func set_item(new_item_data, amount = 1):
    if new_item_data:
        item_data = new_item_data
        item_amount = amount

        # Устанавливаем текстуру
        item_texture.texture = load(item_data.texture_path)
        item_texture.visible = true

        # Показываем количество если больше 1
        if amount > 1:
            amount_label.text = str(amount)
            amount_label.visible = true
        else:
            amount_label.visible = false
    else:
        # Очищаем слот
        clear_slot()

# Очистка слота
func clear_slot():
    item_data = null
    item_amount = 0
    item_texture.visible = false
    amount_label.visible = false

# Получение предмета из слота
func get_item():
    return {"data": item_data, "amount": item_amount}

# Подсветка слота (при наведении или выборе)
func set_highlight(active):
    highlight.visible = active
    if active:
        modulate = Color(1.2, 1.2, 1.2)
    else:
        modulate = Color(1, 1, 1)

# Обработка ввода (перетаскивание)
func _gui_input(event):
    if event is InputEventMouseButton:
        if event.pressed and event.button_index == BUTTON_LEFT:
            # ЛКМ - выбор предмета
            if item_data:
                emit_signal("slot_selected", slot_index)

        elif event.pressed and event.button_index == BUTTON_RIGHT:
            # ПКМ - использование предмета
            if item_data:
                use_item()

Как это работает:

Panel служит контейнером для слота, TextureButton (или Sprite) отображает предмет. Каждый слот хранит данные о предмете и его количестве. Поддержка drag-and-drop через _gui_input().

Особенности:

  • Визуальная подсветка при наведении
  • Отображение количества предметов (если > 1)
  • Обработка ЛКМ и ПКМ отдельно
  • Поддержка перетаскивания между слотами

4. Текстурные кнопки (TextureButton)

Создадим кнопки с разными текстурами для каждого состояния.

Скрипт текстурной кнопки 
extends TextureButton

# Текстуры для разных состояний
export(Texture) var normal_texture
export(Texture) var hover_texture
export(Texture) var pressed_texture
export(Texture) var disabled_texture

# Звуки кнопки
export(AudioStream) var hover_sound
export(AudioStream) var click_sound

# Текст кнопки
export(String) var button_text = "Кнопка"
export(Color) var text_color = Color.white

# Дополнительные эффекты
export var enable_hover_effect = true
export var enable_click_animation = true

onready var label = $Label
onready var audio_player = $AudioStreamPlayer

func _ready():
    # Настраиваем кнопку
    setup_button()

    # Подключаем сигналы
    connect("mouse_entered", self, "_on_mouse_entered")
    connect("mouse_exited", self, "_on_mouse_exited")
    connect("pressed", self, "_on_pressed")

# Настройка кнопки
func setup_button():
    # Устанавливаем текстуры
    texture_normal = normal_texture
    texture_hover = hover_texture if hover_texture else normal_texture
    texture_pressed = pressed_texture if pressed_texture else normal_texture
    texture_disabled = disabled_texture if disabled_texture else normal_texture

    # Настраиваем текст
    if label:
        label.text = button_text
        label.add_color_override("font_color", text_color)
        label.add_font_override("font", load("res://fonts/button_font.tres"))

# При наведении мыши
func _on_mouse_entered():
    if enable_hover_effect and not disabled:
        # Эффект увеличения
        var tween = Tween.new()
        add_child(tween)
        tween.interpolate_property(self, "rect_scale",
            rect_scale, Vector2(1.05, 1.05), 0.1)
        tween.start()

        # Проигрываем звук
        if hover_sound:
            audio_player.stream = hover_sound
            audio_player.play()

# При уходе мыши
func _on_mouse_exited():
    if enable_hover_effect and not disabled:
        var tween = Tween.new()
        add_child(tween)
        tween.interpolate_property(self, "rect_scale",
            rect_scale, Vector2(1, 1), 0.1)
        tween.start()

# При нажатии
func _on_pressed():
    if click_sound:
        audio_player.stream = click_sound
        audio_player.play()

    if enable_click_animation:
        # Анимация нажатия
        var tween = Tween.new()
        add_child(tween)

        # Сначала уменьшаем
        tween.interpolate_property(self, "rect_scale",
            rect_scale, Vector2(0.95, 0.95), 0.05)

        # Потом возвращаем
        tween.interpolate_property(self, "rect_scale",
            Vector2(0.95, 0.95), Vector2(1, 1), 0.1,
            Tween.TRANS_ELASTIC, Tween.EASE_OUT, 0.05)

        tween.start()

Как это работает:

TextureButton имеет 4 состояния с разными текстурами: normal, hover, pressed, disabled. Каждое состояние можно настроить отдельно. Добавлены звуковые эффекты и анимации для лучшего UX.

Ключевые функции:

  • Отдельные текстуры для каждого состояния
  • Анимации при наведении и нажатии через Tween
  • Звуковая обратная связь
  • Поддержка текста на кнопке

5. Панель меню с контейнерами

Создадим главное меню с использованием контейнеров для автоматического расположения элементов.

Скрипт панели меню 
extends PanelContainer

# Элементы меню
onready var title_label = $VBoxContainer/TitleLabel
onready var button_container = $VBoxContainer/ButtonContainer
onready var version_label = $VBoxContainer/VersionLabel

# Кнопки меню
var menu_buttons = []

func _ready():
    setup_menu()
    adjust_layout()

# Настройка меню
func setup_menu():
    # Стилизуем панель
    add_stylebox_override("panel", load("res://styles/menu_panel.tres"))

    # Устанавливаем заголовок
    title_label.text = "Главное меню"

    # Создаём кнопки меню
    create_menu_button("Продолжить", "resume_game")
    create_menu_button("Новая игра", "start_new_game")
    create_menu_button("Настройки", "open_settings")
    create_menu_button("Об игре", "show_credits")
    create_menu_button("Выход", "quit_game")

    # Устанавливаем версию
    version_label.text = "Версия %s" % Global.game_version

# Создание кнопки меню
func create_menu_button(text, action):
    var button = TextureButton.new()
    button.text = text
    button.connect("pressed", self, "_on_menu_button_pressed", [action])

    # Загружаем настройки кнопки
    var button_style = preload("res://scenes/ui/MenuButton.tscn")
    if button_style:
        var button_instance = button_style.instance()
        button_instance.set_text(text)
        button_container.add_child(button_instance)
        menu_buttons.append(button_instance)

    return button

# Обработка нажатия кнопки
func _on_menu_button_pressed(action):
    match action:
        "resume_game":
            hide_menu()
        "start_new_game":
            start_new_game()
        "open_settings":
            open_settings()
        "show_credits":
            show_credits()
        "quit_game":
            quit_game()

# Адаптация под разные разрешения
func adjust_layout():
    var screen_size = get_viewport_rect().size

    if screen_size.y < 600:
        # Маленькое разрешение - уменьшаем отступы
        $VBoxContainer.add_constant_override("separation", 10)
        for button in menu_buttons:
            button.rect_min_size.y = 40
    else:
        # Большое разрешение - стандартные отступы
        $VBoxContainer.add_constant_override("separation", 20)
        for button in menu_buttons:
            button.rect_min_size.y = 50

    # Центрируем меню
    rect_position = (screen_size - rect_size) / 2

6. Управление всем GUI через один скрипт

Создадим централизованную систему управления всеми элементами GUI.

Главный скрипт GUI 
extends CanvasLayer

# Ссылки на все элементы GUI
onready var health_bar = $HUD/HealthBar
onready var coin_label = $HUD/CoinLabel
onready var inventory_panel = $HUD/InventoryPanel
onready var menu_panel = $MenuPanel
onready var tooltip = $Tooltip

# Состояние GUI
var is_menu_open = false
var is_inventory_open = false

func _ready():
    # Инициализируем GUI
    initialize_gui()

    # Подключаемся к глобальным сигналам
    Global.connect("player_health_changed", self, "_on_player_health_changed")
    Global.connect("coins_changed", self, "_on_coins_changed")
    Global.connect("item_collected", self, "_on_item_collected")

# Инициализация всех элементов
func initialize_gui():
    # Скрываем меню при старте
    menu_panel.visible = false
    tooltip.visible = false

    # Настраиваем HUD
    update_health(Global.player_health, Global.player_max_health)
    update_coins(Global.coins)

    # Настраиваем инвентарь
    initialize_inventory()

# Обновление здоровья
func update_health(current, max_health):
    health_bar.max_value = max_health
    health_bar.value = current
    health_bar.get_node("Label").text = "HP: %d/%d" % [current, max_health]

# Обновление монет
func update_coins(amount):
    coin_label.text = str(amount)

# Инициализация инвентаря
func initialize_inventory():
    for i in range(inventory_panel.slot_count):
        var slot = inventory_panel.get_slot(i)
        slot.connect("slot_selected", self, "_on_inventory_slot_selected")

# Показать/скрыть меню
func toggle_menu():
    is_menu_open = !is_menu_open

    if is_menu_open:
        menu_panel.show_menu()
        get_tree().paused = true
    else:
        menu_panel.hide_menu()
        get_tree().paused = false

# Показать/скрыть инвентарь
func toggle_inventory():
    is_inventory_open = !is_inventory_open
    inventory_panel.visible = is_inventory_open

    if is_inventory_open:
        # Пауза игры при открытом инвентаре
        get_tree().paused = true
    else:
        get_tree().paused = false

# Показать подсказку
func show_tooltip(text, position):
    tooltip.text = text
    tooltip.rect_position = position
    tooltip.visible = true

    # Автоматически скрыть через 3 секунды
    yield(get_tree().create_timer(3.0), "timeout")
    tooltip.visible = false

Как это работает:

CanvasLayer содержит все элементы GUI. Централизованный скрипт управляет состоянием интерфейса, обрабатывает ввод и координирует взаимодействие между элементами. Используются сигналы для связи между компонентами.

Преимущества такого подхода:

  • Единая точка управления всем GUI
  • Автоматическая обработка паузы игры
  • Централизованная обработка ввода
  • Лёгкое добавление новых элементов
  • Согласованное поведение всех компонентов

Практические советы

Совет 1: Используйте темы (Themes)

Создайте Theme Resource для единообразия стиля всех элементов GUI. Это упростит изменение дизайна.

Совет 2: Адаптивный дизайн

Используйте Anchors и Margins вместо фиксированных позиций. Это сделает GUI адаптивным под разные разрешения.

Совет 3: Анимации

Добавляйте плавные анимации для всех изменений в GUI. Используйте Tween для плавного перемещения, изменения размера и прозрачности.

Совет 4: Звуковая обратная связь

Добавляйте звуки для всех действий в GUI: наведение, нажатие, получение предмета, изменение значений.

Заключение

Создание качественного GUI в Godot — это не так сложно, как может показаться на первый взгляд. Используя встроенные элементы и следуя приведённым примерам, вы сможете создать интерфейс для своей игры, который будет не только функциональным, но и приятным для игрока.

Помните: хороший GUI должен быть:

  • Интуитивно понятным — игрок сразу понимает, что делать
  • Информативным — показывает всю необходимую информацию
  • Ненавязчивым — не мешает игровому процессу
  • Эстетичным — соответствует стилю игры
  • Отзывчивым — быстро реагирует на действия игрока
