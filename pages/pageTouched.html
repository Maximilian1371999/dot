ТОП 7 скриптов на касание в Godot: от перезагрузки уровня до уничтожения предметов!

15 мин чтения

Взаимодействие

В этой статье мы рассмотрим 10 скриптов, которые срабатывают при касании игроком различных зон. Каждый пример включает код, визуализацию и практическое применение.
Если ещё не читали статью про Area2D, то почитайте. Ибо эта статья - продолжение той темы!

Используйте интерактивные элементы рядом с кодом!
Нажимайте кнопки, чтобы увидеть, как работает логика касания в реальном времени.

Для реализации этих скриптов, вам нужно создать узел Area.
Далее создать внутри него CollisionShape или CollisionPolygon (или их 2D версии) по размерам предмета, которого коснётся игрок.

1. Перезагрузка уровня при касании

Идеально для смертельных зон, лавы или ям. Когда игрок касается такой зоны, уровень перезагружается.

Скрипт перезагрузки уровня 
extends Area2D

# Время перед перезагрузкой, если нужно, чтобы уровень не сразу перезагружался
export var reload_delay = 1.0

func _ready():
    connect("body_entered", self, "_on_body_entered")

func _on_body_entered(body):
    if body.name == "Player":
        print("Игрок коснулся смертельной зоны!")

        # Чтобы было интереснее можно добавить эффекты
        $AudioStreamPlayer2D.play()

        # Активируем таймер
        yield(get_tree().create_timer(reload_delay), "timeout")

        # Перезагружаем текущую сцену
        get_tree().reload_current_scene()

2. Увеличение скорости персонажа

С помощью Area, можно создать зоны ускорения, например, для гоночных игр. При касании с такой зоной скорость игрока временно увеличивается.

Скрипт ускорения персонажа 
extends Area2D

# Множитель скорости
export var speed_multiplier = 1.5
# Длительность эффекта
export var effect_duration = 3.0

func _ready():
    connect("body_entered", self, "_on_body_entered")

func _on_body_entered(body):
    if body.name == "Player":
        var player = body
        var original_speed = player.speed

        # Увеличиваем скорость
        player.speed = original_speed * speed_multiplier

        # Запускаем таймер для возврата
        yield(get_tree().create_timer(effect_duration), "timeout")

Применение: Гоночные трассы, платформеры с секциями ускорения.

  • Игрок входит в зону ускорения
  • Скорость увеличивается в 1.5 раза
  • Эффект длится 3 секунды
  • Возвращается обычная скорость

    • Текущая скорость: 100 единиц

    3. Уничтожение предмета при касании

    Также с помощью Area можно создать объекты, которые разрушаются при прикосновении игрока.

    Скрипт разрушаемого объекта 
    extends Area2D

# Префаб для эффекта разрушения
export(PackedScene) var destruction_effect

func _ready():
    connect("body_entered", self, "_on_body_entered")

func _on_body_entered(body):
    if body.name == "Player":
        # Проигрываем звук разрушения
        $AudioStreamPlayer2D.play()

        # Создаём эффект разрушения
        if destruction_effect:
            var effect = destruction_effect.instance()
            get_parent().add_child(effect)
            effect.global_position = global_position

        # Отключаем коллизию и скрываем
        $CollisionShape2D.disabled = true
        $Sprite.visible = false

        # Удаляем через 2 секунды
        yield(get_tree().create_timer(2.0), "timeout")
        queue_free()

    4. Телепортация между зонами

    Также мы можем создавать порталы для мгновенного перемещения по уровню.

    Скрипт телепорта 
    extends Area2D

# Связанный портал
export(NodePath) var linked_portal_path
var linked_portal = null

# Время перезарядки
export var cooldown = 1.0
var can_teleport = true

func _ready():
    if linked_portal_path:
        linked_portal = get_node(linked_portal_path)

    connect("body_entered", self, "_on_body_entered")

func _on_body_entered(body):
    if body.name == "Player" and can_teleport and linked_portal:
        can_teleport = false

        # Эффект телепортации
        $AnimationPlayer.play("teleport_out")
        yield($AnimationPlayer, "animation_finished")

        # Перемещаем игрока
        body.global_position = linked_portal.global_position

        # Перезарядка
        yield(get_tree().create_timer(cooldown), "timeout")

    5. Изменение гравитации в зоне

    Ещё вы можете создавать антигравитационные зоны или участки с изменённой физикой.

    Скрипт изменения гравитации 
    extends Area2D

# Новая гравитация
export var new_gravity = -200.0
# Исходная гравитация
var original_gravity = 980

func _ready():
    connect("body_entered", self, "_on_body_entered")
    connect("body_exited", self, "_on_body_exited")

func _on_body_entered(body):
    if body.name == "Player" and body.has_method("set_gravity"):
        body.set_gravity(new_gravity)
        $Particles2D.emitting = true
        $AudioStreamPlayer2D.play()

func _on_body_exited(body):
    if body.name == "Player" and body.has_method("set_gravity"):
        body.set_gravity(original_gravity)
        $Particles2D.emitting = false
    Антигравитационная комната
    Игрок
    Зона высокой гравитации

    Эффекты: Маленький прыжок.

    Текущая гравитация: 980 единиц

    Профессиональный совет

    Комбинируйте несколько зон для создания сложных механик.
    Например, зона ускорения + зона низкой гравитации = супер-прыжок!

    6. Открытие/закрытие дверей

    Следующее, что мы рассмотрим - это зоны-триггеры для автоматических дверей.

    Скрипт автоматической двери 
    extends Area2D

# Ссылка на дверь
export(NodePath) var door_path
var door = null

# Направление открытия
export var open_offset = Vector2(0, -64)
var is_open = false

func _ready():
    if door_path:
        door = get_node(door_path)

    connect("body_entered", self, "_on_body_entered")
    connect("body_exited", self, "_on_body_exited")

func _on_body_entered(body):
    if body.name == "Player" and not is_open:
        open_door()

func _on_body_exited(body):
    if body.name == "Player" and is_open:
        close_door()

func open_door():
    is_open = true
    $AnimationPlayer.play("open")

func close_door():
    is_open = false
    $AnimationPlayer.play("close")

    7. Сбор предметов с разными эффектами

    Теперь рассмотрим, как собирать предметы с разными эффектами (действиями).

    Скрипт собираемого предмета 
    extends Area2D

# Тип предмета
export var item_type = 0  # 0=монета, 1=здоровье, 2=звезда
export var value = 1

func _ready():
    connect("body_entered", self, "_on_body_entered")
    update_appearance()

func update_appearance():
    match item_type:
        0:  # Монета
            $Sprite.texture = preload("coin.png")
        1:  # Здоровье
            $Sprite.texture = preload("heart.png")
        2:  # Звезда
            $Sprite.texture = preload("star.png")

func _on_body_entered(body):
    if body.name == "Player":
        apply_effect(body)
        collect()

func apply_effect(player):
    match item_type:
        0:  # Монета
            Global.coins += value
        1:  # Здоровье
            player.heal(value)
        2:  # Звезда
            Global.points += value

func collect():
    $AnimationPlayer.play("collect")
    $CollisionShape2D.disabled = true
    yield($AnimationPlayer, "animation_finished")
    queue_free()
    Типы предметов
    (нажатие на предмет обозначает касание с ним)
    Монета
    Здоровье
    Звезда

    Эффекты от предметов:

  • Монеты: +1 монета
  • Здоровье: +25 HP (но не больше 100)
  • Звезда: +10 очков

    • Статистика игрока:

    Очки: 0
    HP: 25
    Монеты: 0

    Игровой дизайн

    Балансируйте сложность с помощью предметов: слишком много предметов сделают игру очень простой, слишком мало — сложной.

    Заключение

    Area — невероятно гибкий инструмент в Godot, который открывает огромные возможности для создания интерактивности в играх. Эти скрипты показывают основные механики, которые вам понадобятся в большинстве игр.

    Экспериментируйте, комбинируйте разные скрипты и создавайте уникальные игровые механики! Помните, что многие из этих примеров можно дорабатывать и адаптировать под конкретные нужды вашего проекта.

    Профессиональный совет

    Используйте группы (Groups) для более гибкой логики. Например, вместо проверки if body.name == "Player", можно использовать if body.is_in_group("players"). Это позволит вашим скриптам работать с любыми объектами в группе "players".
