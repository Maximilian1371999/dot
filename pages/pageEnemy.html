Привет, будущие геймдизайнеры! Сегодня мы разберем одну из самых важных механик в играх — преследование врагами. Как сделать так, чтобы враги "умно" преследовали игрока, обходили препятствия и создавали интересные игровые ситуации? Давайте разбираться шаг за шагом!

Умные враги делают игру интересной, а глупые — смешной. Наша задача — найти золотую середину!
Интерактивный пример: Простое преследование

Посмотрите, как враг (красный квадрат) преследует игрока (синий круг). Вы можете управлять игроком мышкой!

Часть 1: Базовые принципы преследования

Прежде чем писать код, давайте разберемся, КАК враг должен преследовать игрока. Есть несколько основных подходов:

Метод Как работает Когда использовать
Прямое преследование Враг движется прямо к игроку Простые игры, открытые пространства
Поиск пути (Pathfinding) Враг находит путь вокруг препятствий Лабиринты, уровни с препятствиями
Преследование с предсказанием Враг стремится в точку, где будет игрок Стрелялки, игры с быстрыми врагами
Патрулирование + преследование Враг патрулирует, а при обнаружении преследует Стелс-игры, игры с охраной

Прямое преследование: Плюсы

  • Просто реализовать
  • Мало вычислений
  • Хорошо работает в 2D
  • Идеально для новичков

Прямое преследование: Минусы

  • Застревает в углах
  • Не обходит препятствия
  • Выглядит неестественно
  • Плохо в лабиринтах

Часть 2: Создаем врага с прямым преследованием (2D)

Начнем с самого простого варианта — враг, который всегда движется в сторону игрока.

Enemy (CharacterBody2D)
Sprite2D (текстура врага)
CollisionShape2D (круглая коллизия)
DetectionArea (Area2D для обнаружения игрока)
VisionCone (спрайт конуса зрения, опционально)
Базовый скрипт прямого преследования (2D) 
extends CharacterBody2D

# Настройки врага
@export var speed = 150.0
@export var rotation_speed = 5.0  # Скорость поворота

# Цель для преследования (игрок)
var target: Node2D = null
# Дистанция для начала преследования
@export var chase_distance = 300.0

func _ready():
    # Ищем игрока по имени (убедитесь, что у игрока имя "Player")
    target = get_node("/root/Game/Player")

func _physics_process(delta):
    if target == null:
        return

    # Вычисляем расстояние до цели
    var distance_to_target = global_position.distance_to(target.global_position)

    # Если цель слишком далеко - не преследуем
    if distance_to_target > chase_distance:
        # Можем добавить патрулирование или idle-анимацию
        velocity = Vector2.ZERO
        move_and_slide()
        return

    # Вектор от врага к игроку
    var direction = (target.global_position - global_position).normalized()

    # Плавный поворот в сторону игрока
    var target_rotation = direction.angle()
    rotation = lerp_angle(rotation, target_rotation, rotation_speed * delta)

    # Двигаемся в направлении игрока
    velocity = direction * speed

    # Применяем движение
    move_and_slide()

# Функция для установки цели извне
func set_target(new_target: Node2D):
    target = new_target
Конус зрения врага

В реальных играх враги не должны "видеть" сквозь стены. Давайте создадим зону обнаружения с конусом зрения!

Скрипт зоны обнаружения с конусом зрения 
extends Area2D

# Угол зрения в градусах
@export var vision_angle = 60.0
# Дальность зрения
@export var vision_distance = 200.0
# Ссылка на врага (родитель)
@onready var enemy = get_parent()

func _process(delta):
    # Обновляем визуализацию конуса зрения (если есть)
    update_vision_cone()

func update_vision_cone():
    # Этот код обновляет спрайт конуса зрения
    if has_node("VisionCone"):
        var cone = $VisionCone
        # Настраиваем размер конуса в зависимости от дистанции
        cone.scale = Vector2(vision_distance / 100.0, vision_distance / 100.0)
        # Можно настроить и другие параметры

# Проверка, виден ли объект в конусе зрения
func is_in_vision(target_position: Vector2) -> bool:
    var direction_to_target = (target_position - global_position).normalized()
    var forward_direction = Vector2.RIGHT.rotated(enemy.rotation)

    # Вычисляем угол между направлением взгляда и целью
    var angle_to_target = rad_to_deg(forward_direction.angle_to(direction_to_target))

    # Проверяем, находится ли цель в пределах угла зрения
    if abs(angle_to_target) <= vision_angle / 2:
        # Проверяем расстояние
        var distance = global_position.distance_to(target_position)
        if distance <= vision_distance:
            # Делаем RayCast, чтобы проверить, нет ли препятствий
            var space_state = get_world_2d().direct_space_state
            var query = PhysicsRayQueryParameters2D.create(
                global_position,
                target_position
            )
            query.exclude = [enemy]  # Исключаем самого врага

            var result = space_state.intersect_ray(query)

            # Если RayCast ничего не нашел (кроме цели) - цель видна!
            if result.is_empty() or result.collider == get_node("/root/Game/Player"):
                return true

    return false

# Сигнал при входе тела в Area2D
func _on_body_entered(body):
    if body.name == "Player" and is_in_vision(body.global_position):
        enemy.set_target(body)
        print("Игрок обнаружен!")

Лайфхак: Оптимизация RayCast

Не делайте RayCast каждый кадр для всех врагов! Вместо этого:
1. Используйте таймер для проверки зрения (раз в 0.2-0.5 секунд)
2. Сначала проверяйте расстояние — если игрок далеко, не делайте RayCast
3. Используйте слои физики, чтобы RayCast проверял только стены

Демонстрация поиска пути

Нажмите на поле, чтобы установить препятствие. Враг найдет путь вокруг них!

Умный враг с NavigationAgent2D 
extends CharacterBody2D

@export var speed = 180.0
@onready var navigation_agent = $NavigationAgent2D
@onready var path_update_timer = $Timer

var target: Node2D = null
var current_path: PackedVector2Array = []

func _ready():
    # Находим игрока
    target = get_node("/root/Game/Player")

    # Настраиваем NavigationAgent
    navigation_agent.path_desired_distance = 4.0
    navigation_agent.target_desired_distance = 4.0

    # Настраиваем таймер обновления пути
    path_update_timer.wait_time = 0.5  # Обновляем путь 2 раза в секунду
    path_update_timer.timeout.connect(_update_path)
    path_update_timer.start()

    # Первое обновление пути
    _update_path()

func _physics_process(delta):
    if current_path.size() == 0:
        return

    # Получаем следующую точку на пути
    var next_point = current_path[0]

    # Если близко к точке - переходим к следующей
    if global_position.distance_to(next_point) < 10:
        current_path.remove_at(0)
        if current_path.size() == 0:
            return
        next_point = current_path[0]

    # Двигаемся к следующей точке
    var direction = (next_point - global_position).normalized()
    velocity = direction * speed

    # Плавный поворот
    var target_rotation = direction.angle()
    rotation = lerp_angle(rotation, target_rotation, 5.0 * delta)

    move_and_slide()

func _update_path():
    if target == null:
        return

    # Запрашиваем новый путь к цели
    navigation_agent.target_position = target.global_position

    # Получаем путь как массив точек
    current_path = navigation_agent.get_current_navigation_path()

    # Визуализируем путь (для отладки)
    _draw_path_debug()

func _draw_path_debug():
    # Создаем Line2D для отображения пути
    if not has_node("DebugPath"):
        var line = Line2D.new()
        line.name = "DebugPath"
        line.width = 3.0
        line.default_color = Color(1, 0.5, 0, 0.5)  # Оранжевый полупрозрачный
        add_child(line)

    var line = $DebugPath
    line.points = current_path

    # Автоудаление линии через время
    get_tree().create_timer(0.5).timeout.connect(func():
        if is_instance_valid(line):
            line.queue_free()
    )

# Важно: на сцене должен быть NavigationRegion2D с правильно настроенным NavigationMesh!

Важно про NavigationRegion2D

NavigationAgent не будет работать без NavigationRegion2D!
1. Добавьте NavigationRegion2D на сцену
2. Создайте NavigationMesh (или используйте bake)
3. Убедитесь, что NavigationMesh покрывает всю игровую область
4. Добавьте статические тела (стены) как препятствия

Часть 4: Преследование с предсказанием движения

Этот враг будет стрелять не туда, где игрок сейчас, а туда, где он БУДЕТ!

Враг с предсказанием движения цели 
extends CharacterBody2D

@export var speed = 200.0
@export var prediction_time = 1.0  # На сколько секунд вперед предсказываем

var target: Node2D = null
var target_velocity = Vector2.ZERO
var last_target_position = Vector2.ZERO

func _ready():
    target = get_node("/root/Game/Player")

func _physics_process(delta):
    if target == null:
        return

    # Вычисляем скорость цели (на основе предыдущей позиции)
    if last_target_position != Vector2.ZERO:
        target_velocity = (target.global_position - last_target_position) / delta

    last_target_position = target.global_position

    # Предсказываем будущую позицию
    var predicted_position = target.global_position + (target_velocity * prediction_time)

    # Двигаемся к предсказанной позиции
    var direction = (predicted_position - global_position).normalized()
    velocity = direction * speed

    # Плавный поворот к предсказанной позиции
    var target_rotation = direction.angle()
    rotation = lerp_angle(rotation, target_rotation, 5.0 * delta)

    move_and_slide()

    # Отладочная отрисовка (опционально)
    _draw_prediction_debug(predicted_position)

func _draw_prediction_debug(predicted_pos: Vector2):
    # Рисуем линию к предсказанной позиции
    var canvas = get_canvas_item()
    var start_pos = global_position
    var end_pos = predicted_pos

    # Создаем временный Line2D для отладки
    if not has_node("PredictionDebug"):
        var line = Line2D.new()
        line.name = "PredictionDebug"
        line.width = 2.0
        line.default_color = Color(1, 0, 1, 0.7)  # Фиолетовый
        line.points = [Vector2.ZERO, Vector2.ZERO]
        add_child(line)

    var line = $PredictionDebug
    line.points = [Vector2.ZERO, to_local(end_pos)]

    # Точка в предсказанной позиции
    if not has_node("PredictionPoint"):
        var point = Sprite2D.new()
        point.name = "PredictionPoint"
        point.texture = preload("res://icons/circle.png")
        point.scale = Vector2(0.3, 0.3)
        point.modulate = Color(1, 0, 1, 0.5)
        add_child(point)

    var point = $PredictionPoint
    point.position = to_local(end_pos)

Когда использовать предсказание?

1. Стреляющие враги — чтобы стрелять с упреждением
2. Быстрые погони — чтобы перехватывать игрока
3. Игры с физикой — мячи, снаряды
4. Спортивные симуляторы — передача паса вперед

Часть 5: Конечный автомат (FSM) для сложного поведения

Профессиональные игры используют конечные автоматы для управления поведением врагов.

Визуализация конечного автомата врага

Текущее состояние: PATROL
Враг патрулирует заданную территорию. Ищет игрока.
Конечный автомат для врага 
extends CharacterBody2D

enum State { IDLE, PATROL, CHASE, ATTACK, RETURN }

@export var speed = 150.0
@export var patrol_speed = 80.0
@export var chase_speed = 200.0

var current_state = State.PATROL
var target: Node2D = null
var patrol_points = []
var current_patrol_index = 0
var home_position = Vector2.ZERO

func _ready():
    target = get_node("/root/Game/Player")
    home_position = global_position

    # Создаем точки патрулирования вокруг домашней позиции
    _create_patrol_points()

    # Переходим в начальное состояние
    enter_state(State.PATROL)

func _physics_process(delta):
    match current_state:
        State.IDLE:
            update_idle(delta)
        State.PATROL:
            update_patrol(delta)
        State.CHASE:
            update_chase(delta)
        State.ATTACK:
            update_attack(delta)
        State.RETURN:
            update_return(delta)

    move_and_slide()

# --- IDLE (Ожидание) ---
func enter_idle():
    velocity = Vector2.ZERO
    $AnimationPlayer.play("idle")

func update_idle(delta):
    # Можем добавить случайные повороты или анимации
    pass

# --- PATROL (Патрулирование) ---
func enter_patrol():
    speed = patrol_speed
    $AnimationPlayer.play("walk")

func update_patrol(delta):
    if patrol_points.size() == 0:
        return

    var target_point = patrol_points[current_patrol_index]
    var direction = (target_point - global_position).normalized()
    velocity = direction * speed

    # Поворот в сторону движения
    if direction.length() > 0:
        var target_rotation = direction.angle()
        rotation = lerp_angle(rotation, target_rotation, 3.0 * delta)

    # Достигли точки патрулирования?
    if global_position.distance_to(target_point) < 10:
        current_patrol_index = (current_patrol_index + 1) % patrol_points.size()

        # Иногда делаем паузу в точке
        if randf() < 0.3:  # 30% chance
            change_state(State.IDLE)
            await get_tree().create_timer(randf_range(1.0, 3.0)).timeout
            change_state(State.PATROL)

# --- CHASE (Преследование) ---
func enter_chase():
    speed = chase_speed
    $AnimationPlayer.play("run")

func update_chase(delta):
    if target == null:
        change_state(State.RETURN)
        return

    var direction = (target.global_position - global_position).normalized()
    velocity = direction * speed

    # Поворот к цели
    var target_rotation = direction.angle()
    rotation = lerp_angle(rotation, target_rotation, 5.0 * delta)

    # Проверяем дистанцию для атаки
    if global_position.distance_to(target.global_position) < 50:
        change_state(State.ATTACK)

    # Проверяем, не потеряли ли цель
    if global_position.distance_to(target.global_position) > 400:
        change_state(State.RETURN)

# --- ATTACK (Атака) ---
func enter_attack():
    velocity = Vector2.ZERO
    $AnimationPlayer.play("attack")

    # Наносим урон (если игрок еще рядом)
    if target and global_position.distance_to(target.global_position) < 60:
        if target.has_method("take_damage"):
            target.take_damage(10)

    # После атаки продолжаем преследование
    await get_tree().create_timer(0.5).timeout
    if current_state == State.ATTACK:
        change_state(State.CHASE)

func update_attack(delta):
    pass  # Вся логика в enter_attack

# --- RETURN (Возврат) ---
func enter_return():
    speed = patrol_speed
    $AnimationPlayer.play("walk")

func update_return(delta):
    var direction = (home_position - global_position).normalized()
    velocity = direction * speed

    # Поворот к дому
    if direction.length() > 0:
        var target_rotation = direction.angle()
        rotation = lerp_angle(rotation, target_rotation, 3.0 * delta)

    # Вернулись домой?
    if global_position.distance_to(home_position) < 20:
        change_state(State.PATROL)

# --- Вспомогательные функции ---
func change_state(new_state: State):
    # Выход из текущего состояния
    exit_state(current_state)

    # Вход в новое состояние
    current_state = new_state
    enter_state(new_state)

func exit_state(state: State):
    # Можно добавить логику выхода из состояния
    pass

func enter_state(state: State):
    match state:
        State.IDLE:
            enter_idle()
        State.PATROL:
            enter_patrol()
        State.CHASE:
            enter_chase()
        State.ATTACK:
            enter_attack()
        State.RETURN:
            enter_return()

func _create_patrol_points():
    # Создаем 4 точки патрулирования вокруг домашней позиции
    patrol_points = [
        home_position + Vector2(100, 0),
        home_position + Vector2(100, 100),
        home_position + Vector2(0, 100),
        home_position + Vector2(-100, 0),
        home_position + Vector2(-100, -100),
        home_position + Vector2(0, -100)
    ]

# Сигналы от зоны обнаружения
func _on_detection_area_body_entered(body):
    if body.name == "Player" and current_state != State.CHASE and current_state != State.ATTACK:
        change_state(State.CHASE)

func _on_detection_area_body_exited(body):
    if body.name == "Player" and current_state == State.CHASE:
        change_state(State.RETURN)

Продвинутый лайфхак: Behavior Trees

Для еще более сложного поведения используйте Behavior Trees (деревья поведения):
1. Создайте скрипт с узлами поведения
2. Каждый узел возвращает SUCCESS, FAILURE или RUNNING
3. Комбинируйте узлы в дерево
4. Пример: Sequence([ВидитИгрока?, ДостатчноБлизко?, Атаковать])

Часть 6: Оптимизация для множества врагов

Что делать, если у вас 100 врагов на сцене? Вот секреты оптимизации:

Оптимизированный менеджер врагов 
extends Node

# Группируем врагов для оптимизации
var active_enemies = []
var inactive_enemies = []
var update_interval = 0.1  # Обновляем 10 раз в секунду вместо 60
var time_since_update = 0.0

func _ready():
    # Находим всех врагов на сцене
    var enemies = get_tree().get_nodes_in_group("enemies")
    for enemy in enemies:
        inactive_enemies.append(enemy)
        enemy.set_physics_process(false)  # Отключаем физику у неактивных

func _process(delta):
    time_since_update += delta

    if time_since_update >= update_interval:
        time_since_update = 0.0
        _update_enemies()

func _update_enemies():
    var player = get_node("/root/Game/Player")
    if not player:
        return

    # Оптимизация 1: Обновляем только врагов рядом с игроком
    for enemy in active_enemies:
        if is_instance_valid(enemy):
            var distance = enemy.global_position.distance_to(player.global_position)

            # Если враг слишком далеко - деактивируем
            if distance > 500:
                _deactivate_enemy(enemy)
            else:
                enemy._optimized_update(update_interval)

    # Активируем врагов, которые стали близко
    for i in range(inactive_enemies.size() - 1, -1, -1):
        var enemy = inactive_enemies[i]
        if is_instance_valid(enemy):
            var distance = enemy.global_position.distance_to(player.global_position)
            if distance <= 500:
                _activate_enemy(enemy)
                inactive_enemies.remove_at(i)

func _activate_enemy(enemy):
    enemy.set_physics_process(true)
    enemy.visible = true
    enemy.get_node("CollisionShape2D").disabled = false
    active_enemies.append(enemy)

func _deactivate_enemy(enemy):
    enemy.set_physics_process(false)
    enemy.visible = false
    enemy.get_node("CollisionShape2D").disabled = true
    active_enemies.erase(enemy)
    inactive_enemies.append(enemy)

# В скрипте врага добавляем:
# func _optimized_update(delta):
#     # Упрощенная версия _physics_process
#     # Без сложных вычислений, только необходимое

5 правил оптимизации ИИ

1. Обновляйте реже — не каждый кадр, а раз в 0.1-0.3 секунды
2. Используйте расстояния — не проверяйте далеких врагов
3. Группируйте вычисления — один RayCast для группы врагов
4. Отключайте невидимых — враги за камерой не нуждаются в ИИ
5. Используйте простую математику — избегайте квадратных корней, используйте squared_distance
